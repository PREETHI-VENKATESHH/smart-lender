# 🏦 Smart Lender – AI Loan Approval System

## 📌 Project Overview

Smart Lender is an AI-powered Loan Approval Prediction System that predicts whether a loan application is likely to be approved based on applicant details. The application uses Machine Learning (XGBoost) and provides an interactive web interface built with Streamlit.

---

## 🚀 Features

- Instant loan approval prediction
- Interactive Streamlit web application
- Machine Learning model using XGBoost
- User-friendly interface
- Data preprocessing and visualization

---

## 🛠️ Technologies Used

- Python
- Streamlit
- XGBoost
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Joblib

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/PREETHI-VENKATESHH/smart-lender.git
```

Go to the project folder

```bash
cd smart-lender
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run streamlit_app.py
```

---

## 🤖 Machine Learning Model

This project uses the **XGBoost Classifier** to predict loan approval based on applicant information.
